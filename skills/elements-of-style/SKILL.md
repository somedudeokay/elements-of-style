# Elements of Style

Use this skill to apply Strunk's style rules without loading the full book by default.

## Workflow

1. Read `references/chapter-map.md` to choose the narrowest relevant chapter.
2. Read only the selected file under `references/chapters/`.
3. Apply the rule pragmatically to the user's writing goal.
4. When editing prose, preserve the user's intended meaning and voice; make the smallest changes that improve clarity, force, correctness, or concision.
5. If several rules apply, prioritize meaning and reader comprehension before mechanical conformity.

## Reference Selection

- Punctuation, commas, apostrophes, comma splices, sentence fragments, dangling modifiers: `references/chapters/02-punctuation-and-basic-usage-rules.md`
- Paragraph unity, topic sentences, active voice, positive form, concrete language, concision, parallelism, word order, tense consistency, sentence emphasis: `references/chapters/03-composition-sentence-and-paragraph-principles.md`
- Headings, numerals, parentheses, quotations, references, syllabication, titles: `references/chapters/04-manuscript-form-quotations-and-typography.md`
- Misused words, idioms, vague stock phrases, confused terms, less/fewer, like/as, literally, while, whom, should/would: `references/chapters/05-misused-words-and-usage-glossary.md`
- Spelling conventions and commonly misspelled words: `references/chapters/06-spelling-and-word-forms.md`
- Teaching, drills, exercises, and practice corrections: `references/chapters/07-punctuation-composition-exercises.md`
- Scope, philosophy, and when to bend rules: `references/chapters/01-purpose-scope-and-style-method.md`

The full converted book is available at `references/elements-of-style.md` when a task genuinely needs the whole source.
